Struct coco::epoch::thread::Thread [−] [src]

struct Thread {
    state: AtomicUsize,
    next: Atomic<Thread>,
}

An entry in the linked list of participating threads.

Fields

state: AtomicUsize

The least significant bit is set if the thread is currently pinned. The rest of the bits encode the current epoch.

next: Atomic<Thread>

The next thread in the linked list of participants. If the tag is 1, this entry is deleted and can be unlinked from the list.

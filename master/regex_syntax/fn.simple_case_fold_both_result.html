fn simple_case_fold_both_result(c1: char) -> Result<usize, usize>

The result of binary search on the simple case folding table.

Note that this binary search is done on the "both" table, such that the index returned corresponds to the first location of c1 in the table. The table can then be scanned linearly starting from the position returned to find other case mappings for c1.
